# Dental One Site (Tailwind + SEO)

## Local Setup
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push to GitHub:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPO.git
git push -u origin main
```
2. Go to https://vercel.com, import your repo, and deploy.
